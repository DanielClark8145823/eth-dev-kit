# 以太坊相关的代码模块


文件名：ethereum_wallet_generator.py
功能：离线生成以太坊钱包（地址 + 私钥 + 助记词），区块链入门必备

文件名：simple_blockchain.py
功能：手写极简区块链（带 POW 挖矿、区块哈希、链式验证），GitHub 最受欢迎的演示代码

文件名：erc20_token.sol
功能：标准 ERC20 智能合约（Solidity），发币必备，区块链核心代码

文件名：bitcoin_address_generator.py
功能：离线生成比特币原生地址（P2PKH），密码学 + 区块链结合

文件名：web3_eth_balance_checker.py
功能：Web3 查询以太坊任意地址余额，对接区块链节点


文件名：blockchain_merkle_tree.py
功能：实现默克尔树（比特币 / 以太坊核心数据结构）

文件名：solana_wallet.py
功能：Solana 生态钱包生成，高性能公链必备代码

文件名：p2p_blockchain_node.py
功能：区块链 P2P 节点通信（简化版），体现分布式原理

文件名：nft_erc721.sol
功能：ERC721 标准 NFT 智能合约，Web3 热门技术

文件名：defi_staking.sol
功能：DeFi 质押挖矿合约，区块链金融核心代码
